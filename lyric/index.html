🎵 Lyric Playback

AI 寫下一個字時，Attention 到底看哪裡？——手工設計 playback，讓你看清 prompt 錨定的機制

選擇生成情境

跟隔壁的 🔍 Attention Sandbox 是姊妹作。 Attention Sandbox 展示「AI 讀懂一句話」（分析模式）；這個展示「AI 寫下一個字」（生成模式）。 兩個模式共用「Q·K 匹配 + softmax」的底層機制，只是 Q 換成「下一個要生成的位置」。

生成過程

🔒 PROMPT 永遠掛在上排——所有生成字都能 attention 回望它。 ✨ GENERATED 逐字長出來——當前字用金色高亮，箭頭代表它「往回看誰」。 粗線 = 注意力大、細線 = 小。
